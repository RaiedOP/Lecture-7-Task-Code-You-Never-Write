## Summary
Four real-world projects completed using the "client, not coder" approach —
describing each problem clearly to an AI, verifying every result against a
known fact, and never trusting output that couldn't be checked.

| # | Project | What it solves |
|---|---------|-----------------|
| 1 | [Money Detective](./project-1-money-detective) | Finds leaks (duplicates, subscriptions) in real transaction history |
| 2 | [What's My Grade, Really](./project-2-whats-my-grade) | Calculates true grade using a specific teacher's weighting/drop rules |
| 3 | [The Books Don't Match](./project-3-books-dont-match) | Reconciles a hand-counted total against a messy digital payment record |
| 4 | [Organize the Mess](./project-4-organize-the-mess) | Safely finds duplicate/forgotten files without touching originals |

Each project folder contains:
- The final working script
- A README covering the problem, prompts used, and verification
- Sample/dummy data (real personal data was never uploaded)
- Captured sample run output

> **Privacy note:** All data in this repository (transactions, grades,
> payment records, files) is synthetic/dummy data shaped like the real
> thing, per the assignment's instruction to never commit real sensitive
> data. The verification logic and findings are real; the numbers are not.

See `REPORT.md` for the full write-up covering all four projects.
