# Project 1 — Money Detective

## Objective

The objective of this project is to analyse transaction history using AI-generated Python code. The script identifies recurring subscriptions, duplicate payments, spending categories, and overall expenses.

---

## AI Tool Used

- ChatGPT

---

## Files

- transactions.csv
- money_detective.py
- prompts.txt

---

## Verification

The output was verified by checking:

- Total spending
- Duplicate electricity bill payments
- Recurring subscriptions (Netflix, Spotify, Adobe, Google One)

The calculated values matched the transaction history.
